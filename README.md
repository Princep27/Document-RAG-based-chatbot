# Document-RAG-based-chatbot
This is an AI-powered Document RAG Chatbot that allows users to ask questions from uploaded documents. It uses FAISS for semantic search, SentenceTransformer embeddings, and a Groq LLM to retrieve relevant context and generate accurate, context-aware responses in natural language.

# Steps
- install requirements.txt libs
- set .env YOUR_API_KEY=GROQ_API_KEY
- add data to data.txt file (for RAG)
- run application using streamlit run app.py
